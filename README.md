# Sales Dashboard (Excel)

##  Project Overview
This project analyzes sales data from 2019–2020 using Excel to identify trends,customer behaviorand key busniess insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

##  Key Insights
- Female customers contribute higher spending compared to male customers.
- High spenders are fewer in number compared to low spenders.
